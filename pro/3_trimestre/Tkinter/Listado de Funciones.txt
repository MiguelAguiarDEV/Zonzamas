Funciones:

    .Tk(): crea una nueva ventana de la aplicación.
    .Label(): crea un widget de texto o imagen.
    .Button(): crea un widget de botón.
    .Entry(): crea un widget de entrada de texto.
    .Frame(): crea un contenedor para otros widgets.
    .Canvas(): crea un widget para dibujar gráficos.
    .Menu(): crea un widget de menú.
    .Checkbutton(): crea un widget de casilla de verificación.
    .Radiobutton(): crea un widget de botón de opción.
    .MessageBox(): muestra un mensaje en una ventana emergente.
    .FileDialog(): muestra un cuadro de diálogo para seleccionar archivos.
    .ColorChooser(): muestra un cuadro de diálogo para seleccionar colores.
    .Font(): crea un objeto de fuente.
Métodos:

    .pack(): empaqueta un widget dentro de su contenedor.
    .grid(): coloca un widget en una grilla dentro de su contenedor.
    .place(): posiciona un widget en una ubicación específica dentro de su contenedor.
    .config(): configura las opciones de un widget.
    .bind(): asocia una acción a un evento (por ejemplo, hacer clic en un botón).
    .get(): obtiene el valor actual de un widget (por ejemplo, el texto en un campo de entrada).
    .set(): establece el valor de un widget (por ejemplo, el texto en un campo de entrada).
    .update(): actualiza la interfaz gráfica de usuario (GUI), útil cuando se realizan cambios dinámicos.
    .destroy(): destruye un widget y lo elimina de la GUI.
    .mainloop(): ejecuta el bucle principal de eventos de la GUI.
